You'll master Flexbox and CSS Grid to control where things go on the page, learn CSS animations and transitions to add interactivity, understand positioning, and — crucially — make your entire portfolio **responsive** so it looks great on phones, tablets, and desktops.

## 🚀 Getting Started

1. **Copy the starter files** from `starter-files/` into your working folder
2. Open `index.html` in your browser and `style.css` in your editor
3. Work through the exercises **in order** — each one builds on the previous
4. Use browser DevTools (F12) to inspect and experiment as you go

---

## 📚 Key Concepts Covered

- **Flexbox** — one-dimensional layout (rows and columns)
- **CSS Grid** — two-dimensional layout (rows AND columns)
- **Positioning** — static, relative, absolute, fixed, sticky
- **Transitions** — smooth state changes on hover/focus
- **Keyframe Animations** — `@keyframes`, entrance animations
- **`transform`** — translate, scale, rotate, skew
- **Responsive Design** — viewport, mobile-first, media queries
- **Box shadows, CSS filters, advanced CSS custom properties**

---

## 💡 Tips

- **Test often** — save your file and refresh the browser after each step
- **Use DevTools** — press F12, click the device icon to simulate phone/tablet screens
- **Don't panic** — if something breaks, check for missing semicolons, curly braces, or typos
- **Animate wisely** — only animate `opacity` and `transform` for smooth, GPU-accelerated effects
